# KU-InterviewTool
私が関西大学総合情報学部祭典実行委員会の総務局長として80~100名弱の面接を担当した際、Googleフォームやフォームとリンクしたスプシでは面接の際個人個人の情報が非情に見づらいということで開発したものです。

DBは個人のGoogleスプレッドシートを利用しています。

https://bc2-ny.github.io/KU-InterviewTool/%E9%9D%A2%E6%8E%A5%E7%AE%A1%E7%90%86.html
このリンクにアクセスした際初めに出てくるGASのリンクは、リンクの記入の横のタブにGAS用のコードを置いているのでそれを貼り付けて下さい。

DB側は
id｜name｜kana｜depts｜reason

学籍番号|名前|	希望する局（例:Json方式で記入）[{"name": "常任局", "rank": 1}, {"name": "構成局", "rank": 2}, {"name": "ライブ局", "rank": 3}, {"name": "講演局", "rank": 4}, {"name": "パンフレット局", "rank": 5}, {"name": "模擬局", "rank": 6}]	| 志望動機｜

といった形で記入されています。
GoogleフォームとリンクしたスプシからGeminiにデータを抽出させてこの用に記述させたファイルをエクスポートさせるのが一番てっとり早いです。

＊GASに用意されたコマンドを入力した後「initSheet」という関数を実行する事を忘れないで下さい。
各学部の祭典実行委員会へ、わからないことがあれば私の方までご連絡下さい。私が設定します。
これらは関西大学各学部祭典実行委員会の為開発されたツールです。
